Javascript Fundamentals

Welcome to the JavaScript fundamentals workshop! In this workshop, we'll cover:

  • Data Types
  • Const vs Let
  • Conditional Statements
  • Loops
  • Functions
  • Document Object Model (DOM) Manipulation
  • Events

Data Types

JavaScript has several data types:

const number = 10;
const string = "Hello World";
const boolean = true;
const array = [1, 2, 3];
const object = { name: "John", age: 30 };
const nullValue = null;
const undefinedValue = undefined;

Const vs Let

const and let are used to declare variables in JavaScript:

const numArray = [1, 2, 3, 4, 5];
numArray.push(6); // This is allowed
// numArray = [1, 2, 3, 4, 5, 6]; // This will cause an error

Conditional Statements

Conditional statements control the flow of your program:

if (number > 5) {
  console.log("number is greater than 5");
} else if (number === 5) {
  console.log("number is equal to 5");
} else {
  console.log("number is less than 5");
}

Loops

Loops are used to repeat a block of code:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Functions

Functions are blocks of code designed to perform a particular task:

function greet() {
  console.log("Hello World");
}
greet();

const greet2 = () => {
  console.log("Hello World");
};
greet2();

Document Object Model (DOM) Manipulation

DOM manipulation allows you to interact with the HTML:

const h1 = document.getElementById("h1");
h1.textContent = "Hello World";
h1.style.color = "red";

const button = document.createElement("button");
button.textContent = "Click me";
document.body.appendChild(button);

Events

Events allow you to execute code when a specific action occurs:

button.addEventListener('click', function() {
  document.body.style.backgroundColor = "black";
  document.body.style.color = "white";
});
